# Solo Tabletop Utils
CLI Utils for running solo TTRPGS in Windows. Right now, this is just a shell script for doing command line rolls, but I will likely add more utils in the future.

## Dice Roller
Usage:
```powershell
roll [num rolls] {num dice}d{die size}[kh/kl][num dice to keep] [DC]
```
Num dice and die size are required; all other params are optional.
